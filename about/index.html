À propos

Ce site (fan-made)

Un site vitrine “guide” pour MY HERO ULTRA RUMBLE : roster + fiches personnages + module de builds T.U.N.I.N.G. L’objectif : une UX rapide, lisible mobile, et une vibe comics.

Non-affiliation
Ce projet n’est pas officiel et n’est pas affilié à Bandai Namco / Byking / détenteurs de droits. Il évite d’intégrer des images officielles dans ce ZIP (pour rester clean côté droits).

Structure des données

  • assets/data/roster.json : personnages & variantes (généré depuis ton HOME)
  • assets/data/content/<baseId>.json : texte éditorial par personnage
  • assets/data/tuning/<variantId>.json : builds T.U.N.I.N.G (à remplir)
Pourquoi JSON ?
C’est simple à versionner, et ça se branche facilement depuis un export CSV/Sheets.

Déploiement

  1. Test local : python -m http.server
  2. Dépose le dossier sur GitHub Pages / Netlify / Vercel
  3. Option : nom de domaine + analytics
Formulaire de contact
Le formulaire est en “mailto” (pas de backend). Tu peux brancher Formspree/Netlify Forms si tu veux.

Automatisation (option)

Dans tools/, tu as un petit script pour régénérer assets/data/roster.json à partir d’un CSV “HOME” au même format.
PLUS ULTRA, mais clair
Site vitrine/guide créé à partir de ton fichier TUNING ULTRA RUMBLE - HOME.csv. Les builds T.U.N.I.N.G détaillés se branchent via des fichiers JSON dans assets/data/tuning/.
