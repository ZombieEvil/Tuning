FAQ

Comment je propose un tuning ?
Donne le perso + la variante + la liste des slots/skills/niveaux. Idéalement : un export JSON suivant example.json.
Pourquoi certains builds sont “à remplir” ?
Le fichier HOME fourni liste les persos/variantes mais ne contient pas les réglages de tuning détaillés.
Tu peux ajouter une carte Google ?
Oui. Ici c’est optionnel : colle ton embed Google Maps (ou retire la section).
Google Maps (optionnel)
Remplace ce bloc par un iframe Google Maps (besoin d’URL/embed). Ici on évite de mettre une adresse par défaut.
PLUS ULTRA, mais clair
Site vitrine/guide créé à partir de ton fichier TUNING ULTRA RUMBLE - HOME.csv. Les builds T.U.N.I.N.G détaillés se branchent via des fichiers JSON dans assets/data/tuning/.
